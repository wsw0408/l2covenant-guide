隐私政策

最后更新于 2026-06-30

信息收集与使用目的

本站使用以下第三方服务，可能会收集部分访问数据。我们仅在必要的范围内使用这些数据，用于改善网站体验与内容质量：

  • 百度统计：用于分析网站访问量、页面热门程度和用户地域分布。所有数据均为匿名化处理，无法用于识别个人身份。
  • Giscus（GitHub Discussions）：评论功能基于 GitHub Discussions 实现。发表评论需要登录 GitHub 账号，评论内容、用户名和头像将公开显示在评论区和关联的 GitHub 讨论帖中。
  • GitHub Pages：本站的静态托管服务。GitHub 作为托管方可能会记录访问日志（如 IP 地址、浏览器 UA、访问时间等），用于服务运维和安全防护。
  • Bilibili 视频嵌入：视频板块使用 Bilibili 官方 iframe 播放器。Bilibili 可能会根据用户登录状态记录视频播放行为。

除上述第三方服务依法收集的数据外，本站自身不收集、不存储任何用户的个人信息（包括但不限于姓名、手机号、游戏账号等）。

Cookie 使用

本站不设置任何第一方 Cookie。第三方服务可能会使用 Cookie：

  • 百度统计：使用 Cookie 区分独立访客和追踪访问路径，用户可通过浏览器设置禁用第三方 Cookie。
  • GitHub / Giscus：登录状态由 GitHub 通过 Cookie 维持，不受本站控制。

用户权利

根据《中华人民共和国个人信息保护法》及相关法规，用户享有以下权利：

  • 知情权：你有权了解哪些数据被收集以及如何使用（即本政策所述内容）。
  • 访问与更正权：对于你在 GitHub Discussions 中发布的评论，可通过 GitHub 平台自行编辑或删除。
  • 删除权：如希望删除你在本站评论区的全部留言，请联系站长，我们将协助你在 GitHub Discussions 中移除相关内容。
  • 拒绝追踪权：可开启浏览器的「请勿追踪」（Do Not Track）功能，或安装广告/统计拦截插件。

数据安全

我们承诺不会将任何用户数据出售、交换或提供给任何第三方用于商业目的。评论数据存储于 GitHub Discussions（美国服务器），受 GitHub 隐私声明 约束。本站作为静态网站，不维护用户数据库，因此不存在用户账号密码泄露风险。

第三方服务条款

使用本站即表示你同时同意以下第三方服务的相关条款：

  • 百度统计隐私政策
  • GitHub 隐私声明
  • Bilibili 隐私政策

政策更新说明

本站保留不时修订本隐私政策的权利。当政策发生重大变更（如新增数据收集方）时，我们将在网站首页公告栏发布更新提示，并同步更新本页面的"最后更新于"日期。建议你定期查阅本页面以了解最新政策。若你继续使用本站，即视为同意修订后的条款。
